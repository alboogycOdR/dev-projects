# Portfolio Protection EA - คู่มือการใช้งาน

## 📊 ภาพรวม
Portfolio Protection EA เป็น Expert Advisor ที่ออกแบบมาเพื่อป้องกันพอร์ตการลงทุนด้วยกลยุทธ์หลากหลายและระบบจัดการความเสี่ยงที่แข็งแกร่ง

## 🎯 กลยุทธ์ที่รองรับ (จัดอันดับตามประสิทธิภาพ)

### 1. **Trend Following Strategy** 
- **Win Rate**: 65-75%
- **Risk/Reward**: 1:2
- **จำนวนออเดอร์**: ปานกลาง (10-15 ต่อสัปดาห์)
- **คำอธิบาย**: ใช้ Moving Average 20 และ 50 เพื่อติดตามเทรนด์
- **เหมาะสำหรับ**: ตลาดที่มีเทรนด์ชัดเจน
- **ข้อดี**: กำไรสูงในตลาดเทรนด์, ความเสี่ยงควบคุมได้
- **ข้อเสีย**: ขาดทุนในตลาด sideways

### 2. **Mean Reversion Strategy**
- **Win Rate**: 70-80%
- **Risk/Reward**: 1:1.5
- **จำนวนออเดอร์**: สูง (20-30 ต่อสัปดาห์)
- **คำอธิบาย**: ใช้ RSI และ Bollinger Bands หาจุดกลับตัว
- **เหมาะสำหรับ**: ตลาด sideways หรือ ranging
- **ข้อดี**: Win rate สูง, เทรดบ่อย
- **ข้อเสีย**: ความเสี่ยงในตลาดเทรนด์แรง

### 3. **Breakout Strategy**
- **Win Rate**: 55-65%
- **Risk/Reward**: 1:3
- **จำนวนออเดอร์**: น้อย (5-10 ต่อสัปดาห์)
- **คำอธิบาย**: เทรดเมื่อราคาทะลุ support/resistance
- **เหมาะสำหรับ**: ช่วงข่าวสำคัญหรือการเปลี่ยนแปลงใหญ่
- **ข้อดี**: กำไรสูงเมื่อถูก, Risk/Reward ดี
- **ข้อเสีย**: False breakout บ่อย

### 4. **Scalping Strategy**
- **Win Rate**: 75-85%
- **Risk/Reward**: 1:1
- **จำนวนออเดอร์**: สูงมาก (50-100 ต่อวัน)
- **คำอธิบาย**: เทรดระยะสั้นด้วย EMA 5 และ 13
- **เหมาะสำหรับ**: ตลาดที่มี volatility สูง
- **ข้อดี**: Win rate สูง, กำไรสะสมเร็ว
- **ข้อเสีย**: ต้องดูตลาดตลอดเวลา, spread กระทบมาก

### 5. **Grid Hedging Strategy**
- **Win Rate**: 60-70%
- **Risk/Reward**: 1:0.7
- **จำนวนออเดอร์**: ปานกลาง (15-25 ต่อสัปดาห์)
- **คำอธิบาย**: วางออเดอร์แบบตาข่ายทั้ง buy และ sell
- **เหมาะสำหรับ**: ตลาดที่ไม่มีทิศทางชัด
- **ข้อดี**: ความเสี่ยงกระจาย, กำไรสม่ำเสมอ
- **ข้อเสีย**: ต้องใช้เงินทุนมาก, ความเสี่ยงในเทรนด์แรง

## 🛡️ ระบบป้องกันพอร์ต

### การจัดการความเสี่ยง
- **Maximum Risk per Trade**: 2% ของเงินทุน
- **Daily Loss Limit**: 5% ของเงินทุน
- **Maximum Drawdown**: 10% ของเงินทุน
- **Maximum Open Trades**: 5 ออเดอร์พร้อมกัน

### ระบบ Trailing Stop
- **Trailing Stop**: 1.5% ของราคาเปิดออเดอร์
- **Auto-adjustment**: ปรับ stop loss อัตโนมัติตามกำไร

## 📈 การติดตั้งและใช้งาน

### ขั้นตอนการติดตั้ง
1. คัดลอกไฟล์ `PortfolioProtectionEA.mq5` ไปยังโฟลเดอร์ `MQL5/Experts`
2. คัดลอกไฟล์ `StrategyAnalyzer.mq5` ไปยังโฟลเดอร์ `MQL5/Indicators`
3. Compile ทั้งสองไฟล์ใน MetaEditor
4. Restart MetaTrader 5

### การตั้งค่าพื้นฐาน
```
Portfolio Protection Settings:
- MaxRiskPercent = 2.0
- MaxDailyLoss = 5.0
- MaxDrawdown = 10.0
- MaxOpenTrades = 5
- UseTrailingStop = true

Strategy Selection:
- UseStrategy1_TrendFollowing = true
- UseStrategy2_MeanReversion = true
- UseStrategy3_Breakout = true
- UseStrategy4_GridHedging = false (สำหรับผู้เชี่ยวชาญ)
- UseStrategy5_Scalping = false (ต้องดูตลาดตลอดเวลา)
```

## 📊 การวิเคราะห์ผลงาน

### ใช้ Strategy Analyzer
1. เพิ่ม `StrategyAnalyzer` ลงในชาร์ต
2. ตั้งค่า `AnalysisPeriod = 1000` (จำนวน bar ที่ต้องการวิเคราะห์)
3. เปิด `ShowDetailedStats = true` เพื่อดูรายละเอียด
4. เปิด `ExportToCSV = true` เพื่อส่งออกข้อมูล

### ตัวชี้วัดสำคัญ
- **Win Rate (WR)**: เปอร์เซ็นต์การชนะ
- **Risk/Reward Ratio (RR)**: อัตราส่วนความเสี่ยงต่อผลตอบแทน
- **Profit Factor (PF)**: อัตราส่วนกำไรต่อขาดทุน
- **Sharpe Ratio**: ประสิทธิภาพการลงทุนปรับด้วยความเสี่ยง

## 🎯 คำแนะนำการใช้งาน

### สำหรับผู้เริ่มต้น
1. เริ่มด้วย **Trend Following** และ **Mean Reversion** เท่านั้น
2. ใช้ lot size เล็ก (0.01) ในช่วงทดสอบ
3. ตั้ง MaxRiskPercent = 1% แทน 2%
4. ติดตามผลงานอย่างใกล้ชิดใน 1-2 สัปดาห์แรก

### สำหรับผู้มีประสบการณ์
1. สามารถเปิดใช้กลยุทธ์ได้ 3-4 แบบพร้อมกัน
2. ปรับ MaxRiskPercent ตามความเสี่ยงที่รับได้
3. ใช้ Scalping ในช่วงเวลาที่ดูตลาดได้
4. ทดสอบ Grid Hedging ในบัญชี demo ก่อน

### การจัดสรรพอร์ต (แนะนำ)
- **40%**: กลยุทธ์ที่ดีที่สุดจากการวิเคราะห์
- **30%**: กลยุทธ์อันดับ 2
- **20%**: กลยุทธ์อันดับ 3
- **10%**: ทดสอบกลยุทธ์ใหม่

## ⚠️ ข้อควรระวัง

### ความเสี่ยง
- ไม่มีกลยุทธ์ใดที่รับประกันกำไร 100%
- ตลาด Forex มีความเสี่ยงสูง
- ควรใช้เงินที่เสียได้เท่านั้น

### ข้อจำกัด
- EA ทำงานได้ดีในสภาวะตลาดปกติ
- อาจมีปัญหาในช่วงข่าวสำคัญ (NFP, FOMC)
- ต้องมีการเชื่อมต่ออินเทอร์เน็ตที่เสถียร

## 🔧 การแก้ไขปัญหา

### ปัญหาที่พบบ่อย
1. **EA ไม่เทรด**: ตรวจสอบการตั้งค่า AutoTrading
2. **ขาดทุนต่อเนื่อง**: ลด MaxRiskPercent หรือหยุดใช้ชั่วคราว
3. **เทรดบ่อยเกินไป**: ปิดกลยุทธ์ Scalping
4. **กำไรน้อย**: เพิ่มจำนวนกลยุทธ์ที่ใช้

### การติดต่อสนับสนุน
- ตรวจสอบ log ใน MetaTrader 5
- บันทึกการตั้งค่าที่ใช้
- ระบุปัญหาที่เกิดขึ้นอย่างชัดเจน
