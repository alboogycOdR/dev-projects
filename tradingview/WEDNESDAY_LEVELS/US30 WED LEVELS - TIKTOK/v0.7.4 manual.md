## 1. Overview <a name="overview"></a>

The WED & MIDNIGHT DST System is a sophisticated technical analysis indicator that identifies and visualizes recurring price levels based on two time-based systems:

- **Wednesday SAST Levels**: Captures the high, low, and open prices at 20:30 South African Standard Time (SAST) on the first five Wednesdays of each month
- **Midnight Levels**: Plots the high and low of the 00:00 UTC+2 candle for each day

These levels often act as significant support and resistance zones in price action.

### Key Capabilities:
- Multi-timeframe compatibility (works on any chart timeframe)
- Automatic DST adjustment for accurate time calculations
- Visual wick interaction markers
- Real-time alert system
- Level optimization to reduce chart clutter
- Extensive customization options

## 2. Installation <a name="installation"></a>

1. Open TradingView and navigate to your desired chart
2. Click on the "Indicators" button in the top toolbar
3. Select "Pine Editor" at the bottom of the indicators panel
4. Copy the entire script code
5. Paste it into the Pine Editor
6. Click "Save" and give it a name (e.g., "WED MIDNIGHT System")
7. Click "Add to Chart"

## 3. Core Features <a name="core-features"></a>

### Wednesday SAST Lines
- **What it does**: Identifies price levels from specific Wednesday candles at 20:30 SAST
- **Why it matters**: These levels often become key support/resistance zones
- **Displays**: High, Low, and Open prices for up to 5 Wednesdays per month

### Midnight Levels
- **What it does**: Captures the high and low of the 00:00 UTC+2 candle daily
- **Why it matters**: Midnight levels frequently act as intraday pivot points
- **Displays**: High and Low prices with customizable lookback period

### Wick Interaction Detection
- **What it does**: Places visual markers when price wicks touch any active level
- **Visual feedback**: Markers automatically match the color of the touched level
- **Purpose**: Helps identify potential reversals or confirmations at key levels

### Alert System
- **Real-time notifications**: Alerts when price wicks interact with levels
- **Cooldown feature**: Prevents alert spam with configurable cooldown period
- **Alert types**: Wednesday HIGH/LOW/OPEN and Midnight HIGH/LOW touches

## 5. Alert Configuration <a name="alert-configuration"></a>

### Setting Up Alerts

1. **Add the indicator to your chart**
2. **Right-click on the chart** and select "Add Alert" OR click the alarm clock icon
3. **Configure alert conditions**:
   - Condition: Select "WED & MIDNIGHT DST System"
   - Select "Any alert() function call"
4. **Set alert actions**:
   - Choose notification methods (popup, email, webhook, etc.)
   - Set expiration date if needed
5. **Create Alert**

### Alert Types You'll Receive:
- `WICK TOUCH – [Date] Wednesday HIGH ([Symbol])`
- `WICK TOUCH – [Date] Wednesday LOW ([Symbol])`
- `WICK TOUCH – [Date] Wednesday OPEN ([Symbol])`
- `WICK TOUCH – Midnight HIGH ([Symbol])`
- `WICK TOUCH – Midnight LOW ([Symbol])`

### Alert Best Practices:
- Set cooldown to at least 30 seconds to avoid spam
- Use different notification sounds for different symbols
- Consider disabling alerts during low-liquidity hours

## 6. Usage Tips & Best Practices <a name="usage-tips"></a>

### For Day Traders:
1. **Focus on current week's Wednesday levels** - Disable older Wednesdays
2. **Use tighter Midnight lookback** - 5-7 days maximum
3. **Enable Level Filtering** - Show only 2-3 levels above/below price
4. **Watch for wick interactions** at these levels for entry/exit signals

### For Swing Traders:
1. **Keep all Wednesday levels visible** - Historical levels matter more
2. **Use full 21-day Midnight lookback** - Longer-term levels act as major S/R
3. **Pay attention to confluence** - Where Wednesday and Midnight levels align
4. **Set wider alert cooldowns** - 60+ seconds to reduce noise

### For Position Traders:
1. **Focus on monthly Wednesday levels** - These become major zones
2. **Disable Midnight levels** - Less relevant for longer timeframes
3. **Use thicker line widths** - Make major levels stand out
4. **Track level touches over time** - Multiple touches increase significance

### Chart Optimization:
1. **Start with default settings** and adjust based on your needs
2. **Use Level Filtering on lower timeframes** (15m, 5m) to reduce noise
3. **Disable labels if using multiple indicators** to prevent overlap
4. **Color code by importance** - Brighter colors for more significant levels

### Trading Strategies:
1. **Bounce Trades**: Enter when price wicks into a level and shows rejection
2. **Breakout Trades**: Wait for clean breaks with volume above/below levels
3. **Confluence Zones**: Trade more aggressively where multiple levels cluster
4. **Range Trading**: Use levels as boundaries for range-bound markets

## 7. Troubleshooting <a name="troubleshooting"></a>

### Common Issues & Solutions:

**Issue**: Lines not showing up
- **Solution**: Check that master toggles are enabled in settings
- Ensure you're on an intraday timeframe (not daily/weekly)

**Issue**: Too many lines cluttering the chart
- **Solution**: Enable Level Filtering and reduce the number of visible levels
- Disable older Wednesday levels you don't need

**Issue**: Alerts not working
- **Solution**: Ensure you've created an alert using TradingView's alert function
- Check that "Show Wick Interactions" is enabled
- Verify cooldown period isn't too long

**Issue**: Labels overlapping
- **Solution**: Increase Wednesday Label Offset value
- Reduce label size or disable labels entirely

**Issue**: Wrong time for Wednesday levels
- **Solution**: The indicator automatically adjusts for DST - no action needed
- Levels are captured at 20:30 SAST (adjusted for daylight saving)

### Performance Tips:
- The indicator works best on 15-minute to 4-hour timeframes
- For fastest performance, limit lookback periods and disable unused features
- If experiencing lag, reduce the number of visible levels using Level Filtering
