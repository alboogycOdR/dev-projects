## 📋 **WHAT EACH LEVEL TYPE DOES**

### **A/V Levels (Your Primary Tool)**
- Swing highs (resistance) and lows (support)
- Green solid = Fresh, Red dashed = Unfresh
- Best probability support/resistance zones

### **Gap Levels (Bounce Plays)**
- Consecutive bullish/bearish candles
- Blue = Bullish, Purple = Bearish
- Often become strong S/R zones

### **QM Levels (Advanced)**
- Price moving into previous structure
- Magenta = Fresh, Orange = Unfresh
- Sophisticated entry zones

### **HTF Levels (Context)**
- Same as CTF but from higher timeframe
- Use for bias and confluence
- Dashed style to distinguish from CTF
