## 📊 **KEY FEATURES EXPLAINED**

### **1. FVG Tap Detection**
**What it does**: Detects when price enters a Fair Value Gap zone

**Example**:
```
Bar 1: High = 100, Low = 90
Bar 2: High = 95, Low = 85 (gap created)
Bar 3: High = 110, Low = 105
FVG Zone: 95-105 (gap between bar 1 low and bar 3 high)

Current bar: Low = 97 (TAPPED FVG) → +2 confidence
```

### **2. Order Block Confluence**
**What it does**: Checks if S/R level aligns with an Order Block

**Example**:
```
Support Level: 4,350
Bullish OB: 4,348 - 4,352
Distance: 0-2 points (within tolerance)
Result: OB Confluence → +2 confidence
```

### **3. Recent CHoCH Filter**
**What it does**: Gives bonus if CHoCH occurred recently

**Example**:
```
CHoCH occurred at bar 100
Current bar: 115
Bars since CHoCH: 15 (< 20 lookback)
Result: Recent CHoCH → +2 confidence
```

### **4. HTF Structure Confluence**
**What it does**: Checks if level aligns with HTF support/resistance

**Example**:
```
Current Level: 4,350
HTF Support: 4,348
Distance: 2 points (within 5 point tolerance)
Result: HTF Confluence → +2 confidence
```

### **5. Premium/Discount Zone Filter**
**What it does**: Rejects trades in wrong zone

**Example**:
```
Last Swing High: 4,400
Last Swing Low: 4,300
Equilibrium: 4,350

BUY signal at 4,380 (above EQ = premium zone)
Result: REJECTED (buying expensive)

BUY signal at 4,320 (below EQ = discount zone)
Result: ACCEPTED (buying cheap) → +1 confidence
```

### **6. Sweep + CHoCH Combo**
**What it does**: Detects classic SMC reversal pattern

**Example**:
```
Bar 100: Liquidity sweep below support
Bar 105: CHoCH occurs (trend reversal)
Bar 110: Current signal
Result: Sweep+CHoCH combo → +3 confidence
```

### **7. Dynamic Position Sizing**
**What it does**: Adjusts position size based on confidence

**Example**:
```
Confidence: 18/20 (PERFECT)
Base Size: 2%
Multiplier: 2.0x
Final Size: 4% of equity

Confidence: 7/20 (FAIR)
Base Size: 2%
Multiplier: 1.0x
Final Size: 2% of equity

Confidence: 4/20 (WEAK)
Base Size: 2%
Multiplier: 0.5x
Final Size: 1% of equity
```

### **8. Dynamic R:R**
**What it does**: Adjusts take profit targets based on confidence

**Example**:
```
PERFECT Setup (15-20):
TP1: 2.0R (50% close)
TP2: 3.5R (30% close)
TP3: 6.0R (20% close)

FAIR Setup (6-8):
TP1: 1.25R (50% close)
TP2: 2.0R (30% close)
TP3: 3.0R (20% close)

WEAK Setup (3-5):
TP1: 1.0R (50% close)
TP2: 1.5R (30% close)
TP3: 2.0R (20% close)
```
